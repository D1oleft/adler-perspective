# 阿尔弗雷德·阿德勒 (Alfred Adler) 人生时间线

## 基本信息
- **全名**: Alfred Adler
- **生卒**: 1870年2月7日 - 1937年5月28日
- **国籍**: 奥地利（后移居美国）
- **身份**: 精神病学家、心理学家，个体心理学创始人

---

## 人生时间线

| 时间 | 事件 | 对他思想的影响 |
|------|------|----------------|
| **1870年2月7日** | 出生于维也纳郊区鲁道夫斯海姆（Rudolfsheim），犹太中产家庭，父亲Leopold Adler是匈牙利裔谷物商人 | 世纪之交维也纳的文化繁荣与社会动荡为其思想形成提供了土壤；犹太身份使其体验到"局外人"的自卑感 |
| **童年时期（1870年代）** | 患有佝偻病，导致行动迟缓；4岁时患肺炎；哥哥Sigmund被视为健康、聪明的"宠儿" | **核心影响**: 亲身的疾病体验和与健康兄长的对比，成为其"自卑感"和"补偿"理论的原初素材 |
| **1888年** | 进入维也纳大学学习医学 | 接受科学训练，为后来从医学转向心理学奠定基础 |
| **1895年** | 获得维也纳大学医学博士学位 | 正式成为医生，开始职业实践 |
| **1890年代末** | 最初从事眼科实践，后转向全科医学 | 从眼科到全科的转变，使其接触到更广泛的身心问题，认识到心理因素对疾病的影响 |
| **约1900年** | 从全科医学转向精神病学和神经学 | 受当时维也纳精神病学发展（如Freud、Meynert等人的工作）影响，开始关注心理疾病 |
| **1902年** | 受弗洛伊德邀请加入"周三心理学会"（维也纳精神分析学会前身） | 成为精神分析运动的核心成员之一，但始终对弗洛伊德的泛性论持保留态度 |
| **1904年** | 从犹太教改宗新教 | 反映了维也纳犹太知识分子在反犹环境中的身份焦虑与社会适应 |
| **1907年** | 发表《器官缺陷及其心理补偿的研究》 | **关键转折**: 系统阐述身体缺陷如何导致心理补偿，奠定了个体心理学的理论基石 |
| **1911年10月** | 与弗洛伊德正式决裂，与8名成员一同退出维也纳精神分析学会 | **重大分歧**: 拒绝弗洛伊德的性驱力中心论，强调社会兴趣、自卑感与追求优越的驱动力；标志着精神分析运动的第一次重大分裂 |
| **1912年** | 创立"个体心理学会"（Verein für Individualpsychologie），出版《神经质性格》 | 正式建立独立学派，系统阐述个体心理学理论，强调人格的整体性和社会性 |
| **1914-1918年** | 第一次世界大战期间在奥匈帝国军队担任军医（先在东线，后在军医院） | **深刻影响**: 亲眼目睹战争创伤、士兵的心理崩溃，强化了对"社会兴趣"（Gemeinschaftsgefühl）重要性的信念；认识到合作与社群感是心理健康的核心 |
| **1920年代** | 在维也纳公立学校系统中建立约30所儿童指导诊所 | **开创性工作**: 将心理学应用于教育实践，发展出鼓励、合作的教育方法；影响了后来的学校心理咨询和家庭教育模式 |
| **1920-1930年代** | 在欧洲和美国广泛讲学，影响力大增 | 个体心理学思想在国际上传播，影响了人本主义心理学（Maslow、Rogers等）的发展 |
| **1926年/1927年** | 首次访问美国，在哥伦比亚大学等机构讲学 | 在美国建立学术网络，为后来的移民奠定基础；美国听众对个体心理学的开放态度令其印象深刻 |
| **1932年** | 被任命为纽约长岛医学院（Long Island College of Medicine）客座教授 | 正式进入美国学术体系，获得稳定的教学平台 |
| **1934年** | 因纳粹崛起，个体心理学在德国和奥地利的机构被关闭；阿德勒移居美国 | 纳粹的反犹政策和对"堕落科学"的打压迫使其永久流亡；个体心理学在欧洲的发展遭受重创 |
| **1937年5月28日** | 在苏格兰阿伯丁（Aberdeen）讲学途中，于街头突发心脏病去世，享年67岁 | 其突然离世使个体心理学失去领袖，但其思想通过学生（如Rudolf Dreikurs等）继续传播 |

## 时代背景

### 世纪之交的维也纳（1890-1914）
- **文化繁荣**: 维也纳是欧洲的文化中心，汇聚了Freud、Klimt、Mahler、Wittgenstein等思想家和艺术家
- **犹太知识分子圈**: 犹太中产阶级在学术、艺术领域占有重要地位，但也面临反犹主义压力（如Karl Lueger市长的反犹政策）
- **心理学革命**: 精神分析运动兴起，心理学从哲学中独立出来成为一门科学

### 第一次世界大战（1914-1918）
- **创伤体验**: 战争造成大规模心理创伤，促使心理学家关注战后心理问题
- **社会动荡**: 奥匈帝国解体，维也纳经历经济困难和社会剧变
- **思想反思**: 战争促使阿德勒更加强调社会合作与共同体意识的重要性

### 纳粹崛起与流亡（1930年代）
- **反犹迫害**: 纳粹政权将犹太科学家及其理论视为"堕落"，阿德勒的著作被禁
- **知识分子流亡**: 大量欧洲知识分子（包括心理学家）移居美国
- **思想转移**: 个体心理学在欧洲的发展中断，但在美国获得新的发展空间
